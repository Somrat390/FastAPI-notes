# 📝 Notes Manager API (FastAPI)

A beginner-friendly RESTful API to manage notes using Python and FastAPI.

## 🔧 Features
- ✅ Create a note
- ✅ Read all notes
- ✅ Read a single note
- ✅ Update a note
- ✅ Delete a note

## 🚀 Tech Stack
- Python 3
- FastAPI
- Swagger UI for documentation

## ▶️ Run Locally

```bash
uvicorn main:app --reload

Then open your browser and go to:

http://127.0.0.1:8000 – Home route

http://127.0.0.1:8000/docs – Swagger UI
